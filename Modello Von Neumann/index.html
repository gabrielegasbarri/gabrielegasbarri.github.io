-Riassunto Modello Von Neumann-

1/CPU (Central Processing Unit):

È il cervello del computer e si occupa della gestione generale della macchina.
Esegue istruzioni e controlla il flusso di dati all'interno del sistema, poiche e' l'elemento pricipale di elaborazione di una macchina di Von Neumann.
E'molto complessa e al suo interno possiamo trovare tre componenti principali:
- la memoria cache e i registri
- l'unita' aritmetico-logica
- l'unita dio controllo(Central Unit)

2/Memorie:

  • Memoria Principale (RAM):

    Conserva temporaneamente dati e istruzioni in uso dalla CPU.

  • Memoria Secondaria (es.HDD, SSD):

    Conserva dati a lungo termine anche quando il computer è spento.

3/Bus:

Connessioni fisiche che trasmettono dati tra la CPU, le memorie e le periferiche.

4/Periferiche di Input e Output:

  • Input:

    Tastiera, mouse, scanner, etc.

  • Output:

    Monitor, stampante, altoparlanti, etc.

5/Cache:

Memoria veloce utilizzata per memorizzare temporaneamente dati frequentemente utilizzati dalla CPU, riducendo i tempi di accesso alla memoria principale.
